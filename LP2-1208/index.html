Escrituras condicinais

Condicional simples

Estruturas condicionais servem para alterar o fluxo de execução do meu algoritmo. Após um teste de condição, se a condição for verdadeira, o fçuxo de execução do código interno à estrutural condicional executa. Caso contrário, o código salta para o final da estrutura condicional.

If (Teste de condição){
comandos executados quando a condição é verdadeira }

Condional Composta

If (Teste de condição){
comandos executados quando a condição é verdadeira }else{ comandos executados quando a condição é falsa. }
